Editing the text

Click inside your new text view to make sure it's focused. The currently focused view has a yellow border around it, rather than gray.

Switching to edit mode

Hexerator is a modal editor. The two modes are View and Edit.

You can see the currently active mode at the left side of the bottom panel. Click Edit or press F2 to switch to edit mode.

Editing the text

When edit mode is active, the cursor keys move the edit cursor rather than scrolling the view. You can press the Home key to jump the cursor to the beginning of the view. Now enter whatever text you want for the save name. You can press the Delete key to zero out bytes to properly nul terminate the string.

Saving

Once you are done with your editing, you can click File -> Save to save the file.
